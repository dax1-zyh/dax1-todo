## 简介

本项目基于nodejs和实现了简单的命令行todo List工具

主要使用了commander.js命令行库和Inquirer.js命令行选项卡库进行开发

发布于npm，名称为"dax1-todo“

## 功能

命令关键字 t

| 指令         | 功能                   |
| ------------ | ---------------------- |
| t add 任务名 | 添加任务至任务列表     |
| t clear      | 清空任务列表           |
| t show       | 查看任务列表并进行操作 |
